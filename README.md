# Blackjack Capstone Project

A text-based Python Blackjack game where the player competes against a computer dealer following standard casino rules

## Features

- Unlimited deck size with no jokers  
- Face cards (J Q K) count as 10  
- Aces count as 1 or 11 depending on the hand  
- Player can Hit or Stand  
- Dealer draws automatically until reaching 16 or higher  
- Calculates totals, handles busts, and determines the winner  
- Simple ASCII art logo for visual appeal  

## How to Play

1. Run `blackjack.py` in your terminal  
2. You are dealt two cards and see the dealer's first card  
3. Choose **Hit** to draw a card or **Stand** to end your turn  
4. Dealer plays and the winner is displayed  
5. Play again when prompted  

## Requirements

- Python 3.x  
- `art` module for ASCII logo (optional)  

## Notes

Blackjack only counts on the first two cards

Busts and Ace values are handled automatically

Demonstrates Python basics: loops, conditionals, functions, lists, and randomization
